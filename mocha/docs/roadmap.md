# Mocha Theme Roadmap

## Current

- Catppuccin Mocha top menu bar tint overlay.

## Possible Future Work

- Optional per-screen color accents.
- Optional menu bar visibility hotkey.
- Optional Hammerspoon-controlled helper surfaces that match Mocha, such as alerts or chooser styling.

## Non-Goals

- Patching SystemUIServer.
- Native macOS extensions.
- Private macOS APIs.
- Broad system color mutation through brittle `defaults` writes.
