# WindowSwitcher Roadmap

## Current

- Windows-style individual window switching on `cmd+tab`.
- Current-screen placement.
- Minimized and hidden window inclusion where Hammerspoon exposes those windows.

## Next

- Add optional thumbnail previews if a reliable Hammerspoon-only approach is practical.
- Add per-app grouping as an optional mode.
- Add configurable ordering policies.
- Improve handling for apps that expose untitled but meaningful windows.
